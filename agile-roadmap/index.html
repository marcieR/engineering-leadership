Planning Notes

  • • Grouping: Give two initiatives the same Project / Goal / Team name to stack them as separate bars in one row.
  • • Span: Bars run from Start Month through End Month, inclusive. Leave Start or End blank to show the row as not yet scheduled.
  • • Confidence: Pair this with the Now / Next / Later framework — keep "Now" bars short and precise, and let "Next" or "Later" work span wider ranges to reflect lower confidence.
  • • Color: Use a distinct color per initiative, or per team, so the timeline reads at a glance in a leadership review.
  • • Source of truth: This is a planning visualization, not your sprint tracker — keep it in sync manually as dates shift.
